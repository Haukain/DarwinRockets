A Little Help Captain!

Rules:

  • The rockets will try to find the best path between the start and the end you set.
  • The planets will stop and the black hole attract the rockets to their doom.
  • The best rockets (blue aura) will reproduce between each generation, the worst (red aura) will disappear.
  • Experiment differents routes and configuration to see if and how they will survive.

Controls:

  • You can create continuous generations with the play button, or a defined number the angle bracket button.
  • You can navigate between the generations by using the slider and/or the keyboard arrows.
  • You can sort the rockets of the generation by clicking on the sort button.
  • You can switch between the chart/simulation screen by clicking on the chart/home button.
  • You can see a rocket flight by clicking on it.
